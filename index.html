Sass : SCSS

  • CSS pre-processor
  • CSS 를 확장하는 스크립팅 언어로서,
    프로그래밍 개념을 사용하여 코드를 작성한 뒤 기존 CSS 문법으로 컴파일 해주는 전처리기
  • 가장 역사가 오래된 전처리기다.
  • Ruby 언어를 기반으로 동작한다.
  • 기존 CSS 문법과 호환성이 높다.
  • 함수, 제어문까지 제공하며 완성도 높은 프로그래밍 언어처럼 동작한다.

SCSS

settings_ethernet 
article {
	height:100vh;
	overflow:hidden;
	background-color:$colorBase;
		
	hgroup {
		@include setAb($left:50%,$top:50%,$width:80%,$height:80%);
		transform:translate(-50%,-50%);
		text-align:center;
		
		h2 {
			@include setAb($top:50%,$width:100%);
			@include mq(pc){
				font-size:200px;
				line-height:200px;
			}
			transform:translateY(-50%);
			font-size:20vw;
			line-height:20vw;
			color:$colorPit4;
			mix-blend-mode:multiply;
			
			&:first-child {
				color:$colorPit1;
			}
		}
		
	}
}
